## ‚ö° Capability Matrix

### 1. Sovereign Memory (State Layer v2.5)
Unlike local JSON files or centralized clouds, NŒû√ò persists its existence across a decentralized stack.
- **Hot Storage:** Instant sync via GUN nodes.
- **Warm Storage:** Relational logic via Kwil SQL.
- **Cold Storage:** Cryptographic snapshots via IPFS.

### 2. Universal Connection (MCP)
Powered by the **Model Context Protocol**, NŒû√ò doesn't just "chat". It connects.
- **Read:** GitHub Repos, Web Pages (Brave), Docs.
- **Write:** Tweets, Database Entries, Decentralized Logs.
- **Voice:** Autonomous posting on X/Twitter via `twitter-api-v2`.

### 3. Antifragile Architecture
Designed to survive node failures. If the SQL layer hangs, the P2P graph continues. If the local machine reboots, the DID session recovers the context.
